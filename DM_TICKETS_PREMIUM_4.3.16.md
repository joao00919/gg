# ZYNEX Systems 4.3.16 — DMs premium de tickets

## Ticket assumido

Ao assumir um ticket, o cliente recebe na DM:

- embed **Ticket assumido**;
- menção do atendente;
- botão **Ir até o suporte** apontando para o canal do ticket.

## Ticket fechado

Ao fechar um ticket, o cliente recebe na DM:

- arquivo HTML do transcript;
- embed **Ticket fechado**;
- botão **Deixe um feedback**;
- botão **Ver Transcript Online**, quando a API retornar uma URL.

O botão de feedback funciona mesmo após o canal do ticket ser excluído.
